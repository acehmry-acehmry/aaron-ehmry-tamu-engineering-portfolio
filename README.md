# Formula SAE Electric High Voltage Accumulator

## Overview

Designed and validated the high-voltage accumulator system for Texas A&M Formula SAE Electric during a one-year development cycle.

## My Role

As High Voltage Mechanical Engineer, I was responsible for:

- Accumulator architecture development
- Container and mounting system design
- Structural integration with the chassis team
- Material selection and manufacturability analysis
- FEA validation of accumulator mounts
- Battery subsystem FMEA development
- Thermal management and cooling analysis

## Tools Used

- SolidWorks
- Python
- FEA
- CNC Machining
- Waterjet Cutting
- 3D Printing
- FMEA

## Key Results

- Developed a structurally integrated accumulator system
- Validated mounting structures through analysis and testing
- Successfully passed Formula SAE technical and safety inspections
- Delivered a competition-ready design under a one-year development cycle
